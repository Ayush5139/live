What is Java ?

Java is an object oriented programming language taht was developed by James gosling at Sun Microsystems, which is now owned by Oracle. In today's time jave is one ofthe most diverse language. Java programming language is high level object oriented programming language designed to have as few implementation deprndencies as possible.

Key features of java.

Here are some important features of Java programminglanguage.

  1. Java programing language is designed to build object-oriented applications.
  2. Java is multithreaded language with automatic memory management.
  3. In Java, there is a feature called WORA, i.e write once run anywhere after compilation.
  4. Java code can be run on any computing platform.
